HTML Elements

Table Elements
Tag Name Description Resource
<table> Table This element represents tabular data- the base to create rows and cells See here
<thead> Table Head This element represents a set of table rows, but with indication that they are at the head (top) of a table with information about the table's column See here
<tbody> Table Body This element represents a set of table rows, indicating that they contain the body of (main) data See here
<tr> Table Row This element represents a defined set of rows in a table, that will contain data inside (usually th and/or td elements) See here
<td> Table Data This element represents defined data of a cell of a table (a child to tr elements) See here
<tfoot> Table Foot This element represents the summary of the columns (e.g. a sum of the given numbers in column) See here
<colspan> Column Span This Attribute indicates how many columns the data cell spans or extends See here

CSS Elements

Quick Styling References
Tag Name Description Resource
float Float This property places an element on the left or right side of it's container - remaining as part of a flow See here
margin Margin This property extends the border area to include empty area used to separate the element from it's neighbors (all four sides of an element) See here
padding Padding This property extends the context area (before margin, extends the content on all four sides of an element) See here
overflow Overflow This property sets the desired behavior when the content does not fit in the elements padding box (Visible, hidden, clip, scroll, auto) See here
display: inline-block Inline Block This property allows you to apply both width and height to an element (respecting all margin- most commonly used with list items horizontally) See here
display: inline Inline This property only respects left and right margins/paddings (cannot apply width or height to an element) See here
position: sticky Sticky This property allows elements to stay on the screen when it can, it won't overlap See here
